## Form Validation

### Client-Side Validation (JavaScript)

```javascript
// Real-time validation on blur
document.querySelectorAll('.form-input').forEach(input => {
  input.addEventListener('blur', validateField);
});

// Enable submit button when form is valid
function checkFormValidity() {
  const form = document.getElementById('beta-signup-form');
  const submitBtn = document.getElementById('submit-btn');
  submitBtn.disabled = !form.checkValidity();
}

// Validate individual field
function validateField(event) {
  const field = event.target;
  const errorSpan = document.getElementById(`${field.id}-error`);

  if (!field.validity.valid) {
    errorSpan.textContent = field.validationMessage;
    field.classList.add('error');
  } else {
    errorSpan.textContent = '';
    field.classList.remove('error');
  }

  checkFormValidity();
}
```

### Server-Side Validation

**Required Validations:**
1. Email format validation
2. Email uniqueness check (no duplicate signups)
3. Sanitize all inputs (prevent XSS)
4. Rate limiting (max 5 submissions per IP per hour)
5. CAPTCHA verification (optional, for spam prevention)

## Frequently Asked Questions

### When does the beta start?
Beta invites will be sent in waves starting **December 2025**. Early signups receive priority access. Expect your invite within 1-2 weeks of the first wave.

### What platforms are supported?
Anchor beta is available on:
- **iOS:** iPhone and iPad (iOS 15+) via TestFlight
- **Android:** Phones and tablets (Android 12+) via APK download
- **Web:** Desktop browsers (Chrome, Safari, Firefox, Edge)

### Is my data safe during beta?
Yes! We use industry-standard encryption (TLS 1.3 in transit, AES-256 at rest) and store data on Supabase's secure infrastructure. However, since this is beta software, we **strongly recommend backing up important links** elsewhere. You can export your data anytime from app settings.

### Will I lose my data after beta ends?
No! Your beta account will automatically transition to a production account when we launch. All your saved links, tags, and spaces will be preserved. You'll also get grandfathered pricing (50% off first year) as a thank you for testing.

### How long is the beta period?
The beta will run for **3-4 months** (December 2025 - Q1 2026). We'll notify you at least 2 weeks before beta ends and the public launch happens.

### Will Anchor be free?
Anchor will have a **free tier** with core features (unlimited links, 3 spaces, basic search). Premium features (unlimited spaces, advanced search, team sharing) will be available via subscription ($5/month or $50/year). Beta testers get premium free during beta + 50% off first year.

### Can I share my beta access with friends?
Beta accounts are **non-transferable** and tied to your email address. However, you can refer friends to join the waitlist! For every 3 referrals who sign up, you move up 10 spots in the queue (and they do too!).

### What should I do if I find a bug?
We **love** bug reports! You can report bugs in three ways:
1. **In-App:** Settings ‚Üí Report a Bug (fastest!)

### How do I provide feedback?
We want to hear from you! Send feedback via:
1. **In-App:** Settings ‚Üí Send Feedback

### Can I use Anchor for work/commercial purposes during beta?
Yes, but with caution. Beta software may have bugs or data loss risks, so we recommend using it for **non-critical work** only. For mission-critical links, please maintain backups in another tool until public launch.

You can leave anytime by:
1. Deleting your account in Settings ‚Üí Account ‚Üí Delete Account
