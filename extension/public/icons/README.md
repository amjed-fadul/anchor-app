# Extension Icons

The extension needs icons in the following sizes:

- `icon-16.png` - 16×16px (toolbar icon)
- `icon-32.png` - 32×32px (toolbar icon @2x)
- `icon-48.png` - 48×48px (extensions page)
- `icon-128.png` - 128×128px (Chrome Web Store, large icon)

## Requirements

- **Style:** Simple, clean, recognizable at small sizes
- **Colors:** Black icon on transparent background (matches brand)
- **Format:** PNG with transparency
- **Guidelines:** Follow Chrome Web Store icon guidelines
